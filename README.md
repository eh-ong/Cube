## 1. 프로젝트 개요
<h3>● 주제 및 선정 배경</h3>
  
  > 기존 사내 업무용 웹 페이지는 복잡한 구조와 불필요한 작업 절차로 인해 업무 효율을 저하시키곤 한다.
  > 
  > 이를 개선하기 위해 직관적이고 편리하고 사용자 중심의 서비스를 제공하는 그룹웨어 서비스를 제공하고자 한다.
